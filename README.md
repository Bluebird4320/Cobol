# ⚡ COBOL道場
**COBOLエンジニア向け学習アプリ**

COBOLコーディング・単体テスト・テストケース作成を、Gemini AIを使って楽しく学べるWebアプリです。

## 🚀 使い方

1. **アプリを開く** → https://[username].github.io/cobol-dojo/
2. **Gemini APIキーを入力**
   - [Google AI Studio](https://aistudio.google.com) で無料取得
   - `AIzaSy...` で始まるキーを入力してスタート
3. タブを選んで学習開始！

## 📱 機能

| タブ | 内容 |
|------|------|
| 📚 基礎学習 | COBOL構造・PIC句・命令文・ファイル操作・テスト設計 |
| ❓ 一問一答 | 5択・◯✕・記述 形式のクイズ（AI生成） |
| 💻 コーディング | COBOLコード穴埋め問題（AI生成） |
| 🧪 テスト設計 | テストケース・境界値分析問題（AI生成） |
| 📖 用語集 | COBOL/テスト用語を検索 |
| 💬 AIチャット | わからないことをAIに質問 |
